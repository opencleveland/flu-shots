This is a flu shot finder for Cleveland, OH.  This project is currently in development and not yet complete.

The data behind this site is in [this Google Fusion Table](https://www.google.com/fusiontables/DataSource?docid=16qTOuEkPzhrAAj9-pd9HW9sGNxt54-2SCeyKtShp).

This is [Open Cleveland's](https://github.com/opencleveland) redeployment of the [flu shot finder](https://github.com/tkompare/flushots2013) that [Tom Kompare](https://github.com/tkompare) created for Chicago.
